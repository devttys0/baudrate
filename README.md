# Baudrate Python 3 - fork of baudrate

```bash
git clone https://github.com/sickcodes/python3-baudrate
cd python3-baudrate
pipenv run pip install -r requirements.txt
pipenv run sudo python baudrate.py -a

# pip install -r requirements.txt
```
